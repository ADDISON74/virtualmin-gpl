Automatische Antwort senden
Wenn diese Option aktiviert ist, löst jede E-Mail an diesen Benutzer eine automatische Antwort aus, die den in das nebenstehende Textfeld eingegebenen Text enthält. Der Text kann oben zusätzliche E-Mail-Header enthalten, z. 
Betreff: Ihr Autoreply-Betreff
Cc: foo@bar.com

Dies ist Ihre automatische Antwortnachricht.
