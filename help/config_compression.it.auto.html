Formato di compressione del backup
Per impostazione predefinita, Virtualmin utilizza la compressione gzip nei suoi file di backup. Tuttavia, questa opzione può essere utilizzata per selezionare il formato bzip2 efficiente in termini di spazio (ma ad uso intensivo di CPU). È inoltre possibile utilizzare backup in formato tar non compresso, che possono essere utili se si desidera un backup rapido o se si sta scrivendo su nastro.

Questa opzione viene utilizzata solo durante il backup dei server virtuali in una directory, con un file per dominio. Quando si esegue il backup su un singolo file, il nome file determina il formato di compressione (come domains.tar.gz o domains.tar ).
