Format de compression de sauvegarde
Par défaut, Virtualmin utilise la compression gzip sur ses fichiers de sauvegarde. Cependant, cette option peut être utilisée à la place pour sélectionner le format bzip2 économe en espace (mais gourmand en CPU). Vous pouvez également utiliser des sauvegardes au format tar non compressées, ce qui peut être utile si vous souhaitez une sauvegarde rapide ou si vous écrivez sur bande.

Cette option n'est utilisée que lors de la sauvegarde de serveurs virtuels dans un répertoire, avec un fichier par domaine. Lors de la sauvegarde sur un seul fichier, le nom de fichier détermine le format de compression (comme domains.tar.gz ou domains.tar ).
