Формат сжатия резервной копии
По умолчанию Virtualmin использует сжатие gzip в своих файлах резервных копий. Тем не менее, эта опция может быть использована для выбора вместо bzip2 формата bzip2 с эффективным использованием пространства (но с интенсивным использованием процессора). Вы также можете использовать несжатые резервные копии в формате tar, которые могут быть полезны, если вам нужно быстрое резервное копирование или запись на ленту.

Этот параметр используется только при резервном копировании виртуальных серверов в каталог с одним файлом на домен. При резервном копировании в один файл имя файла определяет формат сжатия (например, domains.tar.gz или domains.tar ).
